# react-portfolio

## Site
https://developergwon-react.netlify.com/

## Spec
- webpack 4.x
- Husky, Prettier, Eslint
- React(16.12.x), Hooks, Redux, Redux-Thunk, Redux-Saga

## Se
- node.js package manager로 yarn을 사용	


- yarn 설치	
```	```
npm install --global yarn
```	

- yarn 실행	

```	
yarn	
```

## Run

```
yarn start
```
